# du-script

## Завдання
Скрипт виводить в порядку спадання дані про об'єм (в байтах) зайнятого дискового простору перших десяти папок/файлів для двох заданих директорій.

## Запуск
```bash
chmod +x top10.sh
./top10.sh /home /var/log
```

## Як працює скрипт
- `du -ab` — показує розмір кожного файлу/папки в байтах
- `sort -rn` — сортує в порядку спадання
- `head -10` — бере перші 10
- `awk` — форматує виведення
